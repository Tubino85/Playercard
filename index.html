ID Nome ATK BLK DEF SET SERV

Suggerimento: premi Invio nell'ID per passare al campo successivo.

Rotazioni (6)

Inserisci l'ID in ciascuna posizione. I blocchi mostrano automaticamente ATK/SET e BLK/DEF/SERV.

Regolamento — bozza compatta

Struttura

  • Ogni squadra compila il Roster e definisce 6 Rotazioni con gli ID.
  • Le statistiche disponibili per ogni giocatore sono: ATK, BLK, DEF, SET, SERV.

Risoluzione azioni (D6)

  • Per ogni gruppo d'azione (es. Attacco vs Muro/Difesa; Servizio vs Ricezione; Alzata), entrambe le parti tirano 1d6 e sommano la relativa statistica.
  • Attacco: ATK (attaccante) + d6 contro BLK+DEF (avversario) + d6. Maggiore vince lo scambio; pareggio = rigiocata.
  • Servizio: SERV (battitore) + d6 contro DEF (ricezione) + d6 → successo = pressione/punto; fallimento = sideout.
  • Alzata: SET (alzatore) può dare un bonus situazionale (+1/+2) all'ATK del finalizzatore.

Rotazioni

  • Si gioca alternando l'attivazione tra le squadre. In caso di punto, la squadra al servizio ruota.
  • Le 6 posizioni di ciascuna rotazione devono essere popolate da ID validi.

Nota: inserisci qui il tuo testo di regolamento integrale quando pronto. Questa è solo una traccia compatta.

Per usare online, carica questo file insieme a players.json nella stessa cartella (GitHub Pages / Netlify / hosting statico).
